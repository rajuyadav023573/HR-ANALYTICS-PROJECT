# HR Analytics – Attrition Analysis (Power BI)

## Project Overview
This project uses Power BI to analyze employee attrition and understand why employees
leave an organization. The dashboard helps HR teams identify attrition trends and make
data-driven decisions to improve employee retention.

## Project Objectives
- Analyze overall employee attrition
- Identify attrition by age group, salary slab, job role, education field, and experience
- Provide insights for HR decision-making

## Tools & Skills Used
- Power BI Desktop
- DAX (Measures and KPIs)
- Data Cleaning and Preparation
- Data Visualization
- HR Analytics

## Key KPIs
- Total Employees
- Total Attrition
- Attrition Percentage
- Average Age
- Average Years at Company

## Key Insights
- Highest attrition occurs in the 26–35 age group
- Employees with lower salary slabs have higher attrition
- Sales Executive and Laboratory Technician roles show higher attrition
- Employees with 0–2 years at the company are more likely to leave

## Project Files
- HR ANALYTICS PROJECT.pbix – Power BI dashboard file
- HR ANALYTICS PROJECT.pdf – Dashboard export

## Conclusion
This Power BI dashboard helps organizations identify high-risk employee groups and
develop strategies to reduce employee attrition.
